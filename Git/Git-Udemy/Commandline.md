## Git command lines

Install git

After install from websites, need to add user.name and user.email
```
git config --global user.name "your name"
git config --global user.email your e-mail
# Best the same with github e-mail
```
git init
```

```
git status
```

Remove git
```
ls -a
```
Then could see .git

Remove git
```
rm -rf .git
```
```

git add

git commit
